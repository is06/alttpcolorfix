PNG2N64
===============================

This is a PNG file converter to RGBA32 binary format for the Nintendo 64.

Dependencies
------------

    sudo apt install libpng-dev

Build
-----

    make

Usage
-----

    png2n64 source.png

This will produce a result.bin file.
