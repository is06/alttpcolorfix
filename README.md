AlttpColorFix
===============================

This is a color fix program that converts a french ALTTP colored png file into true (almost) colors.

Dependencies
------------

    sudo apt install cmake libpng-dev

Build
-----

    cmake .
    make

Usage
-----

    alttpcolorfix source.png

This will produce a result.png file.
